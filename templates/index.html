insert

Create a new asset or license. Leave id blank. Required names must be entered in the correct ID columns from the reference tables.

action,item_type,id,name,asset_tag,model_id,status_id,category_id,seats insert,asset,,Laptop 01,LT001,317,5,, insert,license,,Microsoft 365,,,,,12,10
update

Change an existing record by OCAM id. Fill only the columns you want to change. Use __clear__ to clear an optional value.

action,item_type,id,notes,location_id,product_key update,asset,2044,Moved to HQ,1, update,license,209,,,,AAAA-BBBB-CCCC
delete

Remove an existing asset or license by OCAM id. Other columns are ignored. Preview first because deletion is permanent in OCAM.

action,item_type,id delete,asset,2044 delete,license,209
Required fields for insert
Assets: Asset tag (asset_tag), Model (model_id), and Status (status_id).
Licenses: Software Name (name), Category Name (category_id), and Seats (seats).
Columns ending in _id must use OCAM numeric IDs. Open any reference tile, search by name, then copy the ID into the CSV.
For license product keys, keep using the CSV column product_key; the portal sends it to OCAM correctly during import.
